Apt Repository Status

Each apt repository may contain packages for multiple linux distributions. If a single distribution in the repository has an error, the entire repository will be marked red. Empty repositories are greyed out. Each repository is checked for package checksum mismatches, metadata checksum mismatches, and metadata signing errors.
